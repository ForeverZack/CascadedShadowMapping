# CascadedShadowMapping
Unity内置的方向光动态阴影技术是Cascaded Shadow Mapping(简称CSM)。 由于Unity封装的原因，可能并不能满足实际项目动态阴影的需求，但我们可以从Unity开源出来的built-in shaders看出一些实现细节，这些可以作为自定义实现CSM的参考......
http://blog.chenyong.me/2017/10/09/csm/
